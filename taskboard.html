User Interface Guide

Add Task

Edit Task

Move Task

API Reference

Simulated RESTful endpoints. Base URL: /api

GET /tasks

Fetches the current state of the entire board.

POST /tasks

Creates a new task.

{
                        "column": "backlog",
                        "text": "New Task",
                        "description": "...",
                        "date": "2024-01-20",
                        "priority": "P1"
                        }
PUT /tasks/:id

Updates an existing task.

{
                        "column": "in-progress",
                        "text": "Updated Title",
                        "description": "...",
                        "date": "2024-01-25",
                        "priority": "P2"
                        }
DELETE /tasks/:id

Permanently removes a task.
